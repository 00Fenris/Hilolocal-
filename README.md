# HiloLocal - Proyecto Innovación (Grupo 6)

Este es el repositorio oficial de la landing page de **HiloLocal**, desarrollada para la asignatura de Gestión de Proyectos de Innovación.

## 🚀 Cómo ver la web en vivo (GitHub Pages)

Para que tu profesora pueda ver la web funcionando sin descargar nada, sigue estos pasos:

1. Crea un nuevo repositorio en tu cuenta de GitHub llamado `hilolocal`.
2. Sube todos los archivos de esta carpeta (`index.html`, `style.css` y la carpeta `assets`).
3. En el repositorio, ve a **Settings** (Ajustes) > **Pages**.
4. En "Build and deployment", asegúrate de que esté seleccionado "Deploy from a branch" y elige la rama `main` y la carpeta `/ (root)`.
5. ¡Listo! En unos minutos tendrás un enlace tipo `tuusuario.github.io/hilolocal` que podrás incluir en el Word.
